# repo-manager

## About

* repositoryの管理
* masterブランチをビルドしてnexusにuploadする
* バージョンが被っていた場合マイナーを変更
* ブランチの最新をupload

## Requirement

* java
* kutil-app

# Making

## Version

### Example: 9.1dev

* **9.1dev**
    * バージョン
    * **9**: メジャー
    * **1**: マイナー
    * **dev**: ブランチ/mainのときはなし
